Інструкція: Простий Flask сайт (Windows PowerShell)

Мета
- Показати простий вебсайт на Flask.
- Використати код з попередніх завдань (читання з `4lab.txt` та логіку підрахунку бочок) як вміст однієї зі сторінок.

Файли, що створені
- `flask_app/app.py` — простий Flask-застосунок з маршрутами `/`, `/file`, `/barrels`.
- `flask_app/requirements.txt` — залежності (Flask).

Підготовка (Windows PowerShell)
1. Перейдіть у корінь проекту (припущення: ви у папці `c:\Users\User\Desktop\project1`).

2. Створіть та активуйте віртуальне середовище (PowerShell):

```powershell
python -m venv venv
.\venv\Scripts\Activate.ps1
```

> Якщо політика виконання блокує скрипти, тимчасово дозвольте виконання (як адміністратор):

```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```

3. Встановіть залежності:

```powershell
pip install -r .\flask_app\requirements.txt
```

Запуск застосунку
- Запустіть сервер командою:

```powershell
python .\flask_app\app.py
```

- Відкрийте у браузері: http://127.0.0.1:5000/

Що роблять маршрути
- `/` — головна сторінка з посиланнями.
- `/file` — читає і відображає вміст `4lab.txt` з кореня проекту (UTF-8).
- `/barrels` — використовує логіку з ваших попередніх прикладів (випадкові значення для бочок) та виводить повідомлення і суму.

Поради
- Переконайтесь, що файл `4lab.txt` знаходиться у кореневій папці `c:\Users\User\Desktop\project1` (або змініть шлях у `app.py`).
- Для production використовуйте WSGI-сервер (наприклад, gunicorn) і налаштуйте безпеку.

Якщо хочете, можу:
- Зробити HTML-шаблони окремими файлами у `templates/`.
- Додати просту форму для введення даних користувачем та збереження у `4lab.txt`.
